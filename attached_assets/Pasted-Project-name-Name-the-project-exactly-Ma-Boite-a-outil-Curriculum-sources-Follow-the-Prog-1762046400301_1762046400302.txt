Home page (mandatory look & behavior)
•	Show category tiles all the same size, regardless of label length or content, use different colors.
•	First row tiles (minimum): Conjugaison, Homophones.
•	Add more same-size tiles for: Orthographe, Grammaire, Accords, Ponctuation, Vocabulaire, Types de phrases (you may include more from Alloprof).
•	Uniform design: identical width/height, same typography scale, same padding, same border radius; text wraps, never resizes the tile.
•	Responsive grid: same-size tiles in 2 columns on small screens, 3 on tablets, 4 on laptop/desktop.
•	Each tile opens a section with flashcards and practice.
Flashcards & activities (for every topic)
•	Big, legible modal (covers ~90% of the screen). Only the content area scrolls.
•	Inside each card:
1.	Rule (concise),
2.	2–4 illustrated examples,
3.	Écouter (TTS) with play/pause + speeds 0.8× / 1× / 1.2×,
4.	Practice (MCQ, remplir les blancs, glisser-déposer) with instant feedback.
•	Persist progress locally; show completion checkmarks and “N/total”.
Word-replacement feature (must-have)
•	In example sentences, certain words are underlined and clickable.
•	On click, show “Essayer avec « … »”, temporarily swap the word in the sentence, highlight the swapped token, and display a short hint: “✅ Remplacement possible — bon indice.” Click again (or “Revenir à la phrase”) to restore the original.
•	Start with these pairs (extendable):
est → était, es → étais, sont → étaient, a → avait,
ou → ou bien, où → dans quel endroit (inline hint if swap isn’t natural),
ça → cela, sa → ma, son → mon, ce → le, se → me,
and the mixed set ces / ses / c’est / s’est / sais / sait with a small explanatory bubble (e.g., “c’est = cela est”, “s’est = se + est”, “sais/sait = verbe savoir”).
•	Underline is dotted, slightly offset; color matches grammar category (Verbe=vert, Déterminant=orange, Pronom=bleu, Conjonction=rouge, Adverbe=jaune, Préposition=violet).
•	Add a small legend below cards: “Clique sur les mots soulignés pour tester un remplacement.”
